Importantly, this is one of the few examples of integrable models<\/a>, i.e., <\/em>one with a high degree of symmetry, which greatly reduces its complexity. Using this well-understood model as a starting point, we then push the study into a less-understood regime. We break the high level of symmetries displayed in the XXZ model by adding extra sites that can be occupied by the photons, making the system no longer integrable.
